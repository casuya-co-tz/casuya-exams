### Added

- Initial implementation of casuya-exams repository
- Question bank with CRUD, filtering, search, and random selection
- Category management with hierarchical support
- Tag management with normalization, usage tracking, and suggestions
- Difficulty manager with configurable levels and stats-based calculation
- Exam builder with section management and auto-fill capabilities
- Scheduling system with overlap detection and participant checks
- Session manager with pause/resume, answer submission, and timeout handling
- Grading engine with 8 question type strategies and extensible registry
- Report generator with summary, detailed, individual, and comparative reports
- Certificate generator with verification codes and template system
- Assessment analytics with difficulty index, discrimination index, and performance trends
- Security manager with configurable rules, attempt limits, and proctoring events
- Comprehensive test suite (81 tests)
- Git best practices and CI/CD configuration
